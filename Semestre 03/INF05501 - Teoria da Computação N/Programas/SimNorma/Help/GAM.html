Guia do Ambiente - Menus

 
Esta seção contém a descrição da ação dos itens do menu principal.

Arquivo

  • Novo: inicia um novo projeto vazio. Use esta opção se não desejar mais trabalhar no seu projeto atual.
  • Abrir: lê um projeto gravado em um arquivo.
  • Salvar: grava o projeto.
  • Salvar Como: grava o projeto num arquivo diferente do atualmente associado ao projeto.
  • Sair: fecha o ambiente.

Editar

  • Desfazer: reverte as operações efetuadas mais recentemente.
  • Refazer: repete as operações efetuadas mais recentemente (atualmente, não funcional).
  • Recortar: copia os dados selecionados para a área de transferência e os exclui da área de trabalho.
  • Copiar: copia os dados selecionados para a área de transferência.
  • Colar: substitui os dados selecionados pelo conteúdo armazenado na área de transferência.
  • Limpar Seleção: exclui os dados selecionados.
  • Selecionar Tudo: seleciona todo o programa na área de trabalho.
  • Atualizar: reinterpreta as bibliotecas importadas pelo programa. Use esta opção se uma biblioteca for modificada durante a edição do código do programa.

Visualizar

  • Barra de Ferramentas: exibe/esconde a barra de ferramentas.
  • Barra de Estado: exibe/esconde a barra de status.
  • Macros Disponíveis: exibe/esconde a lista de macros definidas.
  • Registradores: exibe/esconde a lista de registradores.
  • Saída da Execução: exibe/esconde a lista da computação gerada pelo programa.

Modo

  • Modo de Edição: pára a execução do programa e exibe o editor de código.
  • Modo de Fluxograma: monta o programa e o exibe como um fluxograma. Esse modo permite a execução do programa, mas não permite sua edição.
  • Modo de Instruções Rotuladas: monta o programa e o exibe como um conjunto de instruções rotuladas (no formato de lista). Esse modo também permite a execução do programa, mas não permite sua edição.
  • Executar: simula o programa em Norma, produzindo sua computação e indicando as operações realizadas. A execução direta pode ser interrompida a qualquer momento selecionando a opção parar ou pressionando o botão corresponente na barra de ferramentas. Se você desejar alterar a velocidade de execução, tente alterar as opções de execução.
  • Próximo Passo: executa apenas uma instrução. Em seguida, o simulador retorna ao estado de pausa, permitindo o estudo da operação efetuada.
  • Pausa: interrompe a execução direta do programa.
  • Parar: termina a execução do programa e volta à instrução de partida sem alterar o valor dos registradores.
  • Opções da Execução: exibe a janela de configuração de execução.

Ajuda
