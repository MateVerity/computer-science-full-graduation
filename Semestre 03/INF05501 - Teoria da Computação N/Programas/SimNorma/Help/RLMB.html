Referência da Linguagem Monitè - Bibliotecas

 
O recurso mais poderoso da linguagem Monitè é a declaração de macros. Uma vez criadas macros para operações comuns sobre as operações simples, é possível agrupá-las em arquivos de bibliotecas, os quais podem ser importados em programas futuros usando a diretiva importar. Um arquivo de biblioteca é um arquivo com definições de macros (qualquer definição de programa é ignorada). Não há restrições quanto ao nome de um arquivo de biblioteca.

Uma biblioteca pode importar macros de outra biblioteca, assim como macros podem chamar outras macros. Ao compilar o programa, o SimNorma lê todas definições de macros das bibliotecas. No entanto, se uma biblioteca for modificada durante a programação, as mudanças não terão efeito na janela de macros disponíveis até que o usuário selecione Atualizar no menu Editar.

Para mais informações, consulte Referência da Linguagem Monitè - Macros.
