Visão Geral do SimNorma

SimNorma é um ambiente completo de desenvolvimento para a máquina Norma voltado ao estudante de Teoria da Computação interessado em descobrir as capacidades computacionais da máquina Norma. O SimNorma permite escrever o programa e testá-lo de maneira simples, requerendo para isso apenas que o usuário alterne o modo atual de operação na barra de ferramentas superior.

O SimNorma é capaz de simular uma extensão das linguagens definidas anteriormente para a máquina Norma. Esta nova linguagem, que inclui suporte a macros, importação de bibliotecas, descrição de programas em forma iterativa e/ou monolítica e saída de dados em terminal, foi chamada de Monitè (Monolithic and Iteractive Extensions) e é totalmente compatível com os programas anteriormente produzidos para a máquina Norma. Para mais informações, consulte a Referência da Linguagem Monitè.

Após concluído, o programa pode ser visualizado na forma de fluxograma. Um algoritmo heurístico distribui as instruções numa área de trabalho e automaticamente posiciona os conectores de fluxo entre elas, produzindo instantaneamente um diagrama legível e que requer pouca alteração. O programa ainda pode ser visualizado na forma de instruções rotuladas. Por fim, o programador pode iniciar a execução direta ou passo-a-passo do programa; ele ainda pode alternar entre as visualizações conforme achar conveniente e personalizar o comportamento do simulador. Para mais informações, consulte o Guia do Ambiente.

Instalação e Configuração

Configuração mínima:

  • Processador Pentium II 233MHz ou compatível

  • Microsoft Windows 98/Me/2000

  • 16 MB de memória RAM

  • 1 MB de espaço livre em disco

Configuração recomendada:

  • Processador Pentium III 600MHz ou superior

  • Microsoft Windows 2000 ou XP

  • 128 MB de memória RAM

  • 2 MB de espaço livre em disco

Para instalar o SimNorma, faça o seguinte:

  • Encontre a pasta com o programa de instalação do SimNorma;

  • Execute o arquivo SimNorma.exe;

  • Uma janela de apresentação será exibida. Clique em Avançar;

  • Escolha o diretório onde o SimNorma será instalado. Em seguida, clique em Avançar;

  • Selecione o grupo de programas do menu Iniciar onde será colocado o atalho para o SimNorma. Após, clique em Avançar;

  • Verifique suas opções de instalação. Quando terminar, clique em Avançar para iniciar a instalação.

Quando o processo estiver concluído, clique no menu Iniciar, selecione Programas, aponte para o grupo escolhido na instalação (por padrão, SimNorma) e clique no item SimNorma para executar o simulador.

Para configurar as opções de execução, clique no menu Modo e selecione Opções da Execução. Para mais informações sobre as opções de execução, consulte Guia do Ambiente.

Organização de Projetos

Em geral, um programa para a máquina Norma escrita na linguagem Monitè usará macros criadas anteriormente. Estas macros deverão estar organizadas numa biblioteca e, para serem usadas, devem ser importadas pelo programa. Um arquivo de biblioteca é um arquivo de programa comum, mas apenas suas macros são carregadas; qualquer trecho de programa escrito num arquivo que é importando é automaticamente ignorado.

O projeto poderá ainda conter macros específicas. As macros são o recurso mais poderoso da linguagem Monitè, pois permitem a fácil reutilização do código gerado. A linguagem Monitè suporta dois tipos de macro: macro de operação, e macro de teste.

Após a definição das macros, deve estar definido um programa iterativo ou monolítico. As macros definidas no programa e nas bibliotecas importadas podem ser referenciadas nessa área, ou até mesmo no contexto de outras macros. Ao alternar para o modo de execução, as instruções das macros são substituídas localmente, dando origem a um programa monolítico para Norma convencional.
