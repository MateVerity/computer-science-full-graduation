gebalVisualizer Functionalities

File Menu

  • Open: open output files of Spice simulator (.out)
  • Refresh: update the screen by opening the current file again
  • Exit: exit program

Edit Menu

  • Color: change axis, grids and background color
  • Axis: change the referent axis (horizontal). "Time" is the default
  • Show Grids: show/hide grid
  • Show Axis: show/hide axis

Tools Menu

  • gebalEditor: open an text editor with simple functions, like saving, openinig and editing text files
  • Save as JPEG: save visualization area to a JPEG file
  • Manipulate graphs: algebric manipulation of graphs for later visualization, with functions like addiction, subtraction, division, exponential and logarithm
  • Show Report: open gebalEditor with a report of the current file, showing average, maximum and minimum point of each graph and the file path
  • Average: calculate the average of a graph, choosing the time interval to be analysed

Menu Help

  • About: program credits
  • Tutorial Spice: help about how to describe a circuit and simulate on Spice
  • Help: description of all gebalVisualizer´s functions

Others

  • Zoom: with the mouse it is possible selecting a zoom area; the Fit, Next and Prev buttons help the zoom control
  • Graphs: in the right panel is found the graphs´s names and a button to set the color and other to show/hide the graph
  • Coordinates: the Coordinates panel shows the actual position of the cursor; with the mouse buttons it is possible to set/reset a point and see the difference between two points of the graph (if one point is setted)
