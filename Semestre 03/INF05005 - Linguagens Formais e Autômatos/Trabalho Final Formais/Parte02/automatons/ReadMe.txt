   Este arquivo contém o gerenciador de arquivos multifuncional e integrado WinRAR

   Características do WinRAR:

 * O WinRAR introduz um algoritmo de compressão original. Ele fornece
    altas taxas de compressão em arquivos executáveis, bibliotecas de objeto,
    grandes arquivos de texto, etc.

 * O WinRAR oferece um algoritmo de compressão opcional, altamente optimizado
   para dados multimídia.

 * O WinRAR suporta arquivos e compressão até 9,223,372,036,854,775,807 
   bytes em tamanho, sobre 9000PB.  O número de arquivos comprimidos é,
   para todas os propósitos práticas, ilimitado.

 * O WinRAR fornece um suporte completo para arquivos RAR e ZIP 2.0 e pode
   descomprimir arquivos CAB, ARJ, LZH, TAR, GZ, ACE, UUE, BZ2, JAR,
   ISO, Z, 7Z.

 * O WinRAR suporta arquivo de segurança NTFS e floxo de dados.

 * O WinRAR oferece novas facilidades e funções como uma interface gráfico interativo
   utilizando o mouse e menus, bem como uma interface de linha de comandos.

 * O WinRAR fornece a funcionalidade de criar arquivos 'sólidos',
   que poderão aumentar a taxa de compressão entre 10% - 50% em adição aos
   métodos mais comuns, particularmente quando se comprime um grande número
   de pequenos arquivos.

 * O WinRAR oferece a habilidade de criar e alterar arquivos SFX utilizando
   módulos por padrão e externos.

 * O WinRAR oferece a habilidade de criar arquivos multi-volume como SFX.

 * O WinRAR oferece um número de funções de serviço, tais como colocar uma
   palavra-chave (senha), adicionar ao arquivo e comentários. Até os arquivos
   fisicamente danificados poderão ser reparados e um arquivo poderá ser
   fechado para prevenir eventuais modificações.
